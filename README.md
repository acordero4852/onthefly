# On The Fly

On the Fly is a collaborative trip planner for creating trips and adding destinations, built with a React frontend backed by an Express + PostgreSQL API.

## Tech Stack

- **Frontend:** React, React Router, Vite
- **Backend:** Node.js, Express, Passport
- **Database:** PostgreSQL (`pg`)

## Project Structure

```txt
onthefly/
├── client/   # React + Vite frontend
└── server/   # Express + PostgreSQL backend
```

## Getting Started

### Server

```bash
cd server
npm install
npm run reset   # seed the database
npm start
```

### Client

```bash
cd client
npm install
npm run dev
```
